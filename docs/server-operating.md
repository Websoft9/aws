# 操作

## 启动、停止和终止

在一些特殊情况下，用户打算将VM恢复到初始状态，但希望保留所有VM的配置选项和关联资源均保留。这个时候，我们就需要用到“重新部署”的操作。

## Resize


The VM size can be adjusted, login AWS Portal, and then open: Settings -> Size, click the "Resize" button.
